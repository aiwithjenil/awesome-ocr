# GLM-4V

GLM-4V is a versatile vision-language model from THUDM. It is highly capable in image processing and OCR, supporting high-resolution inputs and providing advanced text extraction from complex images, including charts and handwritten text.

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
python glm4v_ocr.py <path_to_image>
```
