# GNU Ocrad

GNU Ocrad is an OCR program based on a feature extraction method. It reads images in PBM, PGM, or PPM formats and produces text in 8-bit or UTF-8 formats.

## Installation

- **Ubuntu**: `sudo apt install ocrad`
- **macOS**: `brew install ocrad`

## Usage

```bash
python ocrad_ocr.py <path_to_image>
```
