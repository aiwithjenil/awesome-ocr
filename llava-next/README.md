# LLaVA-NeXT

LLaVA-NeXT (LLaVA-v1.6) is an improved version of the Large Language-and-Vision Assistant. It features better reasoning, OCR, and world knowledge compared to its predecessors.

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
python llava_next_ocr.py <path_to_image>
```
