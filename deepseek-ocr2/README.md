# DeepSeek-OCR2

DeepSeek-OCR2 is the next generation of vision-language models from DeepSeek-AI, featuring significant improvements in "Contextual Optical Compression" and document understanding. It is designed to read documents with human-like accuracy and is officially supported in vLLM.

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
python deepseek_ocr2.py <path_to_image>
```
