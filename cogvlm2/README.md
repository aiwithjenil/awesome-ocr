# CogVLM2

CogVLM2 is a powerful vision-language model from THUDM (Tsinghua University). It uses a "Visual Expert" architecture to deeply integrate visual and linguistic information, making it exceptionally strong at OCR and document understanding without external OCR tools.

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
python cogvlm2_ocr.py <path_to_image>
```
