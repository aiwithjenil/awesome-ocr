# VILA (Visual-Language Models for Edge Devices)

VILA is a family of efficient and powerful vision-language models developed by NVIDIA and MIT. It is designed to perform well on a variety of multimodal tasks, including OCR, while being efficient enough for edge deployment.

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
python vila_ocr.py <path_to_image>
```
