# TrOCR (Transformer-based Optical Character Recognition)

TrOCR is an end-to-end Transformer-based OCR model from Microsoft. It uses a pre-trained Vision Transformer (ViT) as the encoder and a pre-trained BERT-like model as the decoder.

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
python trocr_ocr.py <path_to_image>
```
