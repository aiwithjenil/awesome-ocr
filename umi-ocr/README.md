# Umi-OCR

Umi-OCR is an open-source, multi-language OCR software based on PaddleOCR. It is designed for ease of use and features a powerful GUI and an HTTP API for integration into other applications.

## Installation

1. Download and run the Umi-OCR server from the [official repository](https://github.com/hiroi-sora/Umi-OCR).
2. Install Python dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage

```bash
python umi_ocr.py <path_to_image>
```
