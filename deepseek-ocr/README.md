# DeepSeek-OCR

DeepSeek-OCR is a powerful vision-language model developed by DeepSeek-AI, optimized for document understanding and text extraction. It features "Contextual Optical Compression" to handle high-resolution images efficiently.

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
python deepseek_ocr.py <path_to_image>
```
