# GOT-OCR2.0 (General OCR Theory 2.0)

GOT-OCR2.0 is a unified, end-to-end OCR model with 580M parameters. It handles diverse OCR tasks, including plain text, formatted documents, tables, charts, and mathematical formulas.

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
python got_ocr.py <path_to_image>
```
