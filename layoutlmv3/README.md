# LayoutLMv3

LayoutLMv3 is a multi-modal Transformer for Document AI. It pre-trains text and image representations jointly and is highly effective for tasks like document classification, information extraction, and visual question answering.

## Installation

1. Install Tesseract OCR (required for text extraction):
   - **Ubuntu**: `sudo apt install tesseract-ocr`
2. Install Python dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage

```bash
python layoutlmv3_ocr.py <path_to_image>
```
