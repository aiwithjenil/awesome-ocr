# olmOCR

olmOCR is an LLM-based OCR tool from the Allen Institute for AI (AI2) designed to convert document scans into clean, structured text. It excels at removing artifacts and preserving the logical flow of documents.

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
python olmocr_ocr.py <path_to_pdf>
```
