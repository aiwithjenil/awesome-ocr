# MiniCPM-V

MiniCPM-V is a series of efficient end-side multimodal LLMs (MLLMs) developed by OpenBMB. It is designed to be resource-efficient while maintaining high performance on image understanding and OCR tasks.

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
python minicpm_v_ocr.py <path_to_image>
```
